# Premium Nuxt 3 Website

A visually stunning, responsive landing page built with Nuxt 3, Vue 3, and modern CSS.

## Features
- **Modern HSL Dark Theme**: Deep black backgrounds with vibrant blue and violet accents.
- **Glassmorphism**: Elegant translucent components with backdrop-blur effects.
- **Animations**: Smooth fade-ins and floating background blobs.
- **Responsive Design**: Works perfectly across mobile and desktop.
- **Nuxt 3 Power**: File-based routing, SSR, and premium typography.

## Getting Started

### 1. Install Dependencies
```bash
npm install
# or
yarn install
```

### 2. Run Development Server
```bash
npm run dev
# or
yarn dev
```

### 3. Build for Production
```bash
npm run build
```

## Project Structure
- `pages/`: Contains the main routes (`index.vue`, `about.vue`).
- `components/`: Reusable UI components (`AppNavbar.vue`, `HeroSection.vue`).
- `assets/css/`: Global styles and animations (`main.css`).
- `layouts/`: Core page layout (`default.vue`).
